# Meal Mania
**Meals** application. Using the [MealDB API](https://www.themealdb.com/).



 # Tools & Technologies
 * MVI architecture (**MVICore**)
 * RxJava 2
 * Retrofit
 * Hilt
 * Android Architecture Components (Room, View Binding)
 * Material Design
 * Glide
 * Firebase Dynamic Links & Deep links
